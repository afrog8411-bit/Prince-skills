# glm-vision v2：交互式视觉对话 + 智能增强

## 目标

从"一次性 CLI 调用工具"进化为"可交互的视觉助手"。用户丢一张图进来，可以来回聊天、追问、对比，不用每次都输命令。

## 核心新增：`chat` 交互对话模式

```
python glm.py chat <图片路径/URL>
```

进入交互模式后：
1. 首次加载图片，智谱给出全景描述
2. 用户可无限次追问
3. 自动判断用户问题类型，路由到不同模型
4. 维护会话上下文，支持连续对话
5. 输入 `exit` / `quit` 退出

### 智能路由

| 用户问题 | 识别方式 | 路由到 | 说明 |
|----------|---------|--------|------|
| "描述这个"、"这是什么" | 含"描述/什么/是" | glm-4v-flash 直出 | 简单识别 |
| "分析这个"、"有什么问题"、"对比" | 含"分析/问题/对比/评价" | glm-4.1v-thinking-flash + deepseek | 深度分析 |
| "左上角那个"、"那个红色的是什么" | 含空间/颜色/位置词 | glm-4v-flash 聚焦问 | 局部追问 |

### 自动重试

API 429/500/502 错误时自动重试，指数退避（1s → 2s → 4s → 8s），最大 4 次。

### 内容审核友好提示

API 返回 1301 内容审核错误时，检测具体触发关键词，给出建议（"提示词可能触发了安全审核，请避免包含敏感词汇"）。

### URL 支持

`vision`, `chat`, `draw` 命令的图片参数支持 HTTP/HTTPS URL 自动下载。
