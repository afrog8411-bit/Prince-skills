# skill-downloader

**不用出终端，搜索安装 Claude Code Skill。**

从 15 个仓库浏览、搜索、下载安装 Claude Code Skill。支持的仓库包括官方、Composio 社区、宝玉中文等。

## 用法

在 Claude Code 里说：

- "帮我找个能翻译的 skill"
- "搜索能做 PPT 的 skill"
- "看看宝玉有什么 skill"
- "装一个做数据分析的 skill"

## 支持的仓库（15 个）

| 仓库 | 来源 | Skill 数 |
|------|------|----------|
| anthropics/skills | 官方 | 17 |
| ComposioHQ/awesome-claude-skills | 社区 | 140+ |
| JimLiu/baoyu-skills | 中文 | 22 |
| GarethManning/education-agent-skills | 教育 | 131 |
| 其他 12 个仓库 | 社区/个人 | 若干 |
